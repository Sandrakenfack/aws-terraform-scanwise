Résumé du Projet

Réalisation d'un dossier technique complet pour la migration de l'infrastructure IT. Ce projet inclut une veille technologique active, l'évaluation de solutions innovantes (IA et Automatisation) et la sécurisation des données sensibles.

Configuration du Serveur AD et DNS

Projet Technique : Mise en place et configuration d'un serveur Active Directory et DNS.

Systèmes & Réseaux Active Directory DNS

Résumé du Projet

Configuration complète d'un serveur Active Directory (AD) et d'un serveur DNS pour une infrastructure réseau sécurisée. Ce projet couvre l'installation, la configuration et les tests de fonctionnement.

  • Configuration AD : Changement du nom de la machine, attribution d'adresse IP, installation et promotion du contrôleur de domaine.
  • Configuration DNS : Intégration dans le domaine, installation du rôle DNS et création des zones directes et inversées.
  • Tests et Validation : Vérification du domaine et des transferts de zone.
"La configuration précise des serveurs AD et DNS est essentielle pour une infrastructure réseau fiable et sécurisée."
